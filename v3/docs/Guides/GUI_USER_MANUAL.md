## 🎯 主要機能別ガイド

### 1️⃣ **フィルタパネル** - 動画を探す

#### 📝 タイトル検索

**目的**: キーワードで動画を検索

```
使い方:
1. 「タイトル」欄にキーワードを入力
2. 一覧が自動で絞り込まれる
3. 複数キーワードはスペース区切り（例: 「新作 日本」）
```

**使用例**:

| キーワード | 結果 |
|:--|:--|
| `新作` | タイトルに「新作」を含む動画のみ表示 |
| `実況` | 「実況」を含む動画のみ表示 |
| `企画` | 「企画」を含む動画のみ表示 |

#### 📊 投稿状態で絞り込み

**目的**: 投稿済み/未投稿で分類

```
ドロップダウン: [全て ▼]
  ↓
選択肢:
  - 全て: すべての動画を表示
  - 未投稿: 投稿していない動画のみ
  - 投稿済み: 投稿済みの動画のみ
```

**使用シーン**:
- 「未投稿」を選択 → 投稿すべき動画を一覧で確認
- 「投稿済み」を選択 → 今までの投稿履歴を確認

#### 🌐 配信元で絞り込み

**目的**: YouTube/Niconico で分類

```
ドロップダウン: [全て ▼]
  ↓
選択肢:
  - 全て: すべての動画を表示
  - YouTube: YouTube の動画のみ
  - Niconico: ニコニコの動画のみ
```

**使用シーン**:
- 「YouTube」を選択 → YouTube動画だけを投稿
- 「Niconico」を選択 → ニコニコ動画だけを投稿

#### 🔄 リセット

**目的**: すべてのフィルタをクリア

```
使い方:
1. 「🔄 リセット」ボタンをクリック
2. 全フィルタが解除される
3. すべての動画が表示される
```

### 2️⃣ **動画一覧** - 動画を選択・確認

#### ☑️ チェックボックス（選択）

**目的**: 投稿対象の動画を選択

```
使い方:
1. 動画行の左側のチェックボックスをクリック
2. ☑️ マークが表示される
3. 複数選択可能
```

**複数選択の例**:
```
☑ 新作動画1
☑ 新作動画2
☐ 古い動画
☑ 新作動画3

→ 3つの動画が選択状態
```

#### 📋 タイトル列

動画のタイトルを表示。長いタイトルは省略表示。

#### 📊 投稿状態列

| 表示 | 意味 |
|:--|:--|
| **未投稿** | 投稿していない（投稿可能） |
| **投稿済み** | 既に Bluesky に投稿済み |
| **予約中** | スケジュール投稿の予約済み |

#### 🌐 配信元列

| 表示 | 意味 |
|:--|:--|
| **YouTube** | YouTube の動画 |
| **Niconico** | ニコニコの動画 |

#### 📅 公開日時列

動画が公開された日時を表示

**形式**: `2025-12-19 14:30:00`

### 3️⃣ **操作パネル** - ボタン機能

#### 📤 [投稿実行]

**目的**: 選択した動画を Bluesky に投稿

```
使い方:
1. 動画を1つ以上選択（☑ マーク）
2. 「投稿実行」ボタンをクリック
3. ログウィンドウに進捗が表示される
4. 「✅ 投稿完了」と表示されたら成功
```

**確認事項**:
- 選択した動画数を確認
- ネットワーク接続を確認
- PREVENT_DUPLICATE_POSTS 設定を確認

#### 🧪 [投稿テスト]

**目的**: 投稿をシミュレーション（実際には投稿しない）

```
使い方:
1. 動画を1つ選択
2. 「投稿テスト」ボタンをクリック
3. ログウィンドウに処理が表示される
4. 「🧪 投稿テスト成功」と表示されたら成功
```

**用途**:
- 設定が正しいか確認
- テンプレートが正しく機能するか確認
- Bluesky の認証情報が正しいか確認

**重要**: テストなので Bluesky に投稿されません

#### 🗑️ [削除]

**目的**: 選択した動画をデータベースから削除

```
使い方:
1. 削除したい動画を選択
2. 「削除」ボタンをクリック
3. 確認ダイアログが表示される
4. 「OK」をクリックして削除

⚠️ 削除すると復元できません
```

**削除後の動作**:
- データベースから削除される
- 一覧から消える
- 再度 RSS 取得されても削除リストに登録される

#### 🔄 [更新]

**目的**: YouTube/Niconico RSS を手動で再取得

```
使い方:
1. 「更新」ボタンをクリック
2. ログに「RSS取得中...」と表示される
3. 新着動画があれば一覧に追加される
```

**自動更新との関係**:
- 通常は `POLL_INTERVAL_MINUTES` ごとに自動実行
- 「更新」ボタンで強制実行可能

#### ⚙️ [設定]

**目的**: アプリケーション設定を開く

```
使い方:
1. 「設定」ボタンをクリック
2. 設定ウィンドウが開く
3. 各設定項目を変更
4. 「保存」をクリック

詳細は『設定ウィンドウ』セクション参照
```

#### ❌ [終了]

**目的**: アプリケーションを終了

```
使い方:
1. 「終了」ボタンをクリック
2. 処理中の場合は終了待機
3. ウィンドウが閉じる
```

### 5️⃣ **統計情報** - 投稿状況を把握

**場所**: 画面最下部

```
📈 統計情報

総動画数: 42件
  ├─ 投稿済み: 28件 (66%)
  └─ 未投稿: 14件 (34%)

```

**項目説明**:
- **総動画数**: データベースに保存されている全動画
- **投稿済み**: Bluesky に投稿済みの動画
- **未投稿**: まだ投稿していない動画
- **配信元別**: YouTube とニコニコの内訳

## 📖 実践シナリオ

### シナリオ 1: 「新作」動画だけを投稿したい

```
1. フィルタ - タイトル欄に「新作」と入力
2. フィルタ - 投稿状態を「未投稿」に変更
3. 動画一覧に「新作」タイトルで未投稿の動画が表示される
4. 全動画にチェック（☑）を入れる
5. 「投稿実行」ボタンをクリック
6. ログで投稿完了を確認

✅ 完了！
```

### シナリオ 2: YouTube だけ投稿済みを確認

```
1. フィルタ - 配信元を「YouTube」に変更
2. フィルタ - 投稿状態を「投稿済み」に変更
3. YouTube の投稿済み動画だけが表示される
4. 投稿状況を確認

✅ 完了！
```

### シナリオ 3: 投稿テストを実行したい

```
1. 動画を1つ選択（☑）
2. 「投稿テスト」ボタンをクリック
3. ログウィンドウを確認

成功ログ:
  ✅ 投稿テスト成功（本投稿なし）
  ✅ 設定は正しいです

失敗ログ:
  ❌ 認証失敗: Bluesky パスワードが不正です
  ⚠️ 設定を確認してください

✅ テスト完了！
```

## 🎛️ 設定ウィンドウ

[設定] ボタンをクリックすると、以下の設定ウィンドウが開きます。

### 基本設定タブ

| 項目 | 説明 | 初期値 |
|:--|:--|:--|
| **ポーリング間隔（分）** | RSS 更新間隔 | 10 |
| **動作モード** | selfpost / autopost / dry_run / collect | selfpost |
| **デバッグ有効** | DEBUG ログを表示するか | false |

### Bluesky タブ

| 項目 | 説明 |
|:--|:--|
| **ハンドル** | Bluesky ユーザー名（@除く） |
| **パスワード** | アプリパスワード |
| **投稿有効** | Bluesky 投稿を有効にするか |

### プラグインタブ

| 項目 | 説明 |
|:--|:--|
| **有効なプラグイン一覧** | ロード済みプラグインを表示 |
| **プラグインの有効/無効** | チェックボックスで制御 |

## 💡 操作のコツ

### ✅ 効率的な検索方法

```
複数キーワード検索:
- タイトル欄に「新作 実況」と入力
  → タイトルに「新作」と「実況」の両方を含む動画を表示

投稿履歴確認:
- 投稿状態を「投稿済み」に設定
  → 既に投稿した動画を一覧で確認

複合検索:
- タイトル: 「ライブ」
- 投稿状態: 「未投稿」
- 配信元: 「YouTube」
  → YouTube のライブ関連で未投稿の動画
```

### ⚡ 大量投稿の手順

```
1. フィルタで対象動画を絞り込む
2. 絞り込まれた動画が全て表示されることを確認
3. 画面上部のチェックボックスをクリック（全選択）
4. 「投稿実行」をクリック
5. ログで完了を確認

💡 「投稿テスト」で先に確認すると安全
```

### 🔒 重複投稿の防止

```
settings.env で PREVENT_DUPLICATE_POSTS=true に設定
  ↓
同じ動画を投稿しようとしたとき
  ↓
⚠️ 「この動画は既に投稿済みです」と警告
  ↓
投稿が自動中止

→ 誤った重投稿を防止
```

## 🐛 よくあるエラーと対応

### ❌ 「フィルタが反応しない」

**対応**:
1. アプリを再起動
2. 「🔄 リセット」をクリック
3. 再度入力

### ❌ 「ログウィンドウにエラーが表示される」

**対応**:
1. ログの内容を確認
2. [FAQ_TROUBLESHOOTING_BASIC.md](./FAQ_TROUBLESHOOTING_BASIC.md) を参照
3. 該当する対応方法を実行

### ❌ 「投稿できない」

**対応**:
1. 「投稿テスト」で確認
2. ログでエラーメッセージを確認
3. Bluesky の認証情報が正しいか確認
