# YouTube新着動画フィード取得→通知bot(またはPythonアプリ)

## 求める機能
### 複数配信サービス対応
- Twitch（配信開始・終了）
- YouTube（配信開始・終了、新着動画投稿）
- ニコニコ動画（新着動画投稿）

### Blueskyへの自動投稿
- 各サービスごとに投稿内容テンプレートを切り替え可能
(放送開始と終了、動画投稿時でそれぞれ別の内容で投稿できます)
- 画像添付もサポート（1枚まで）
- 投稿履歴はCSVで自動記録

### 柔軟な利用要件
- 必要なのはネット回線とパソコンとアカウントだけ
（ドメインとか固定IPとかは不要）（PythonとGitはPCに導入してください）
- Twitchとの通信はトンネル通信を利用
（ドメインあればCloudflare、なければngrok/localtunnelが使えます）
Twitch以外のサービスはトンネル不要、RSSで取得可能
- Blueskyのアカウントは必須
Twitch/YouTube/ニコニコのいずれかのアカウントが必要
（ニコニコ以外はAPIキーも必要）
- アプリ画面でもコマンドでも全機能利用可能(設定はアプリ画面から)
- アプリ画面はダークモード対応

## 技術仕様
- ニコニコ動画の新着取得はRSSから
- YouTube動画の取得はRSSから
- YouTubeLiveの検知もできればRSSから、必要に応じてAPIを使うが、
API制限が厳しいのでほぼ使わない設計にしたい
- PubSubHubbub や　RSS　で取得・収集して、収集情報をソースとして通知させたい
- 通知はDiscord・及びAPI経由でのBlueskyを検討。
