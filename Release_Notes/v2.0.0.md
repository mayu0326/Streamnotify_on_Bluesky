# Streamnotify on Bluesky - v2.0.0 リリースノート

**リリース日**: 2025年12月16日

## 新機能

### プラグインアーキテクチャ
- プラグインを利用した拡張性の高い設計を導入。
- 主な機能:
  - プラグインの自動検出とロード
  - プラグインごとの設定管理

### Bluesky 投稿機能
- YouTube の新着動画を Bluesky に自動投稿。
- 主な機能:
  - 投稿テンプレートのカスタマイズ
  - 投稿履歴の管理

### SQLite データベース
- 動画情報を管理するためのローカルデータベースを導入。
- 主な機能:
  - 動画の重複チェック
  - 投稿状態の管理

## 改善点
- コードベースをリファクタリングし、モジュール化を強化。
- ログ出力を改善し、デバッグ情報を詳細化。

## バグ修正
- 特定の条件で Bluesky 投稿が失敗する問題を修正。
- YouTube RSS フィードの取得ロジックを最適化。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
