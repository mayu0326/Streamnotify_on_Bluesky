# Streamnotify on Bluesky - v3.0.0 リリースノート

**リリース日**: 2025年12月17日

## 新機能

### マルチテンプレート対応
- YouTube Live/Archive/新着、Niconico新着に対応したテンプレートを追加。
- 主な機能:
  - サービス別・イベント別のテンプレート自動選択
  - テンプレートフォールバック機能（未設定時の自動適用）

### 高度なフィルタリング
- タイトル、配信元、投稿状態、コンテンツタイプでのフィルタリングを追加。

### YouTube 優先度ベースの重複排除
- 新動画 > アーカイブ > 通常動画の優先度で重複を自動排除。

### AssetManager による自動テンプレート配置
- プラグイン導入時に必要なテンプレート・画像を自動配置。

## 改善点
- テンプレートシステムの安定性を向上。
- プラグイン管理の効率化。

## バグ修正
- 特定の条件でテンプレートが正しく適用されない問題を修正しました。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
