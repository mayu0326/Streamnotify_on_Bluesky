# Streamnotify on Bluesky - v3.3.0 リリースノート

**リリース日**: 2025年12月19日

## 新機能

### 一括投稿スケジュール機能
- 複数動画を選択して15分ごと等の分散投稿が可能になりました。
- 主な機能:
  - 複数動画の一括選択
  - スケジュール間隔の設定（5分～60分）
  - スケジュール確認・編集・キャンセル機能
  - Bluesky API レート制限対策

## 改善点
- スケジュール管理のUIを最適化し、操作性を向上しました。

## バグ修正
- スケジュール設定時に発生していた特定のクラッシュを修正しました。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
- スケジュールデータは `data/schedules/` ディレクトリに保存されます。
