# Streamnotify on Bluesky - v3.1.0 リリースノート

**リリース日**: 2025年12月18日

## 新機能

### GUI フィルタ・検索機能
- タイトル検索、投稿状態フィルタ、配信元フィルタ、種類フィルタを追加。
- 主な機能:
  - タイトル検索（リアルタイム検索対応）
  - 投稿状態フィルタ（Posted/Unposted）
  - 配信元フィルタ（YouTube/Niconico）
  - コンテンツタイプフィルタ（🎬 動画、📹 アーカイブ、🔴 配信）

### 重複投稿防止オプション
- `PREVENT_DUPLICATE_POSTS` 設定を追加。
- 同じ動画の再投稿を防止し、誤操作を防ぎます。

### バックアップ・復元機能
- ZIP形式でDB・テンプレート・設定を一括エクスポート/インポート可能。
- 主な機能:
  - API キー・パスワードの安全な除外オプション
  - 復元時の自動バックアップ（既存ファイル保護）
  - 画像フォルダのオプション包含

## 改善点
- GUI の操作性を向上。
- フィルタリング処理のパフォーマンスを最適化。

## バグ修正
- 特定の条件でフィルタが正しく適用されない問題を修正しました。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
