# Streamnotify on Bluesky - v2.3.0 リリースノート

**リリース日**: 2025年12月18日

## 新機能

### YouTube Live プラグイン
- YouTube Live の開始・終了を自動検知し、Bluesky に投稿可能。
- 主な機能:
  - ライブ配信の開始・終了通知
  - アーカイブ動画の自動投稿
  - テンプレート選択機能（yt_online_template.txt / yt_offline_template.txt）

### ニコニコ動画プラグイン
- ニコニコ動画の新着動画を自動検知し、Bluesky に投稿可能。
- 主な機能:
  - RSS フィードを利用した新着動画の検知
  - 投稿テンプレートのカスタマイズ

## 改善点
- SQLite データベースのロック処理を最適化。
- Bluesky API のエラーハンドリングを強化。

## バグ修正
- 特定の条件で投稿が失敗する問題を修正。
- ニコニコ動画のユーザー名取得ロジックを改善。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
