# Streamnotify on Bluesky - v2.1.0 リリースノート

**リリース日**: 2025年12月17日

## 新機能

### 画像自動リサイズ機能
- 投稿画像を自動的にリサイズし、Bluesky の要件に適合。
- 主な機能:
  - 縦横比に基づくリサイズ（16:9、4:5、1:1）
  - JPEG 品質の最適化

### DRY RUN 機能
- 投稿をシミュレートし、実際に投稿せずに動作確認が可能。
- 主な機能:
  - 投稿内容のプレビュー
  - ログにシミュレーション結果を記録

### 投稿設定ウィンドウ
- GUI で投稿設定を簡単に変更可能。
- 主な機能:
  - 画像添付の有無を選択
  - リンクカードの表示設定

## 改善点
- ログ出力を強化し、デバッグ情報を詳細化。
- AssetManager のテンプレート配置処理を最適化。

## バグ修正
- 特定の条件で画像が正しくリサイズされない問題を修正。
- 投稿設定ウィンドウでのクラッシュを修正。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
