# Streamnotify on Bluesky - v3.2.0 リリースノート

**リリース日**: 2025年12月19日

## 新機能

### フィルタ条件の保存機能
- GUI で設定したフィルタをプロファイルとして保存可能になりました。
- 主な機能:
  - フィルタプロファイルの作成・管理
  - 複数プロファイルの保存・切り替え
  - デフォルトプロファイルの自動適用
- 操作効率が向上し、頻繁に使用するフィルタ設定を簡単に再利用できます。

## 改善点
- 内部コードの最適化により、フィルタリング処理のパフォーマンスが向上しました。

## バグ修正
- 特定の条件でフィルタが正しく適用されない問題を修正しました。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。
- フィルタプロファイルは `data/filters/` ディレクトリに保存されます。
